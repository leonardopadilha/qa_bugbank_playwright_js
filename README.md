## Descrição do Projeto

Este projeto envolve a automação de processos utilizando o [bugbank](https://bugbank.netlify.app/), que é um site criado para automação de testes.


## Instalação

Instalar as dependências

```bash
  npm install
```

## Execução dos testes via terminal

```bash
  npm run test
```

## Execução dos testes via interface do playwright

```bash
  npm run test:ui
```
